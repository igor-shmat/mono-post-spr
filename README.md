# Mono Post Api

### Подключение к базе данных

| Параметр               | Значение                      |
|------------------------|-------------------------------|
| Name                   | mono_post                     |
| Login                  | mono_post_user                |
| Password               | mono_post_pass                |
| HOST                   | localhost                     |
| PORT                   | 12345                         |

Для поднятия базы данных необходимо запустить docker-compose.yml

### Работа с Api
Postman коллекция находится внутри проекта в json формате
